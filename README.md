# QtUIharjoituksia
Työpöytäsovellusten luominen QT Designer -ohjelmalla ja Python-kielellä. Harjoituksessa luodaan seuraava käyttöliittymä, jonka avulla harjoitellaan tiedoston käsittelyä, kuvan muokkausta ja tulostamista.
